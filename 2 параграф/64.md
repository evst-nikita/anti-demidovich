№ 64

$$
\lim_{n \rightarrow \infty} {
	\frac{
		\log_a{n}
	}
	{
		n
	}
} = 0 \quad (a > 0)
$$

\medskip

Рассмотрим случай, когда $a = 2$. То есть предел будет:
$$
\lim_{n \rightarrow \infty} \frac{\log_2{n}}{n} = 0
$$
Выполним замену переменных: $k = \log_2{n}$. Тогда $n = 2^k$. Получим предел:
$$
\lim_{k \rightarrow \infty} \frac{k}{2^k} = 0
$$
Полное решение именно этого предела приведено в решении номера 58.
Докажем для остальных $a$. Сделаем это через переход к логарифму с основанием двойки и вынос константы ($\dfrac{1}{\log_2{a}}$).
$$
\lim_{n \rightarrow \infty} \frac{\log_a{n}}{n} = \lim_{n \rightarrow \infty} \frac{1}{\log_2{a}} \cdot \frac{\log_2{n}}{n} = \frac{1}{\log_2{a}} \cdot \lim_{n \rightarrow \infty} \frac{\log_2{n}}{n} = 0
$$

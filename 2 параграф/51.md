\lim_{n \rightarrow \infty}{
	\left(
		\frac{1}{n^2} +
		\frac{2}{n^2} + \cdots +
		\frac{n - 1}{n^2}
	\right)
}
$$
\medskip

Используем формулу суммы геометрической прогрессии:
$$
\lim_{n \rightarrow \infty}{
	\left(
		\frac{1}{n^2} +
		\frac{2}{n^2} + \cdots +
		\frac{n - 1}{n^2}
	\right)
} =
\lim_{n \rightarrow \infty}{
	\left(
		\dfrac{
			\dfrac{(1 + n - 1) \cdot (n - 1)}{2}}{n^2}
	\right)
} =
\lim_{n \rightarrow \infty}{
	\frac{
		n \cdot (n - 1)
	}
	{
		2n^2
	}
} =
$$
$$
= \lim_{n \rightarrow \infty}{
	\frac{
		n^2 - n
	}
	{
		2n^2
	}
} = \lim_{n \rightarrow \infty}{
	\frac{
		(n^2 - n) : n^2
	}
	{
		2n^2 : n^2
	}
} = \lim_{n \rightarrow \infty}{
	\frac{
		1 - \dfrac{1}{n}
	}
	{
		2
	}
} = \frac{1}{2}
$$
Напомню, что $\frac{1}{n} \rightarrow 0$ при $n \rightarrow \infty$.

$$
\lim_{n \rightarrow \infty}{\frac{10000n}{n^2 + 1}}
$$

\medskip

$$
\lim_{n \rightarrow \infty}{\frac{10000n}{n^2 + 1}} = 10000\lim_{n \rightarrow \infty}{\frac{n}{n^2 + 1}}
$$
$$
x_n = \frac{n}{n^2 + 1} \quad 0 \leq \frac{n}{n^2 + 1} < \frac{n}{n^2} = \frac{1}{n}
$$
По теореме о двух жандармах или первому признаку сущ. предела в предисловии параграфа, получаем, что, так как:
$$
\lim_{n \rightarrow \infty} 0 = \lim_{n \rightarrow \infty}{\frac{1}{n}} = 0 \quad \mbox{и} \quad 0 \leq \frac{n}{n^2 + 1} \leq \frac{1}{n}
$$
То
$$
\lim_{n \rightarrow \infty}{\frac{n}{n^2 + 1}} = 0
$$
Итого:
$$
\lim_{n \rightarrow \infty}{\frac{10000n}{n^2 + 1}} = 10000\lim_{n \rightarrow \infty}{\frac{n}{n^2 + 1}} = 10000 \cdot 0 = 0
$$

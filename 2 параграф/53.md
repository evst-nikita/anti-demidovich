\medskip

Используем в решении формулу суммы квадратов, доказанную в №2:
$$
\lim_{n \rightarrow \infty}{
	\left[
		\frac{1^2}{n^3} +
		\frac{2^2}{n^3} +
		\cdots +
		\frac{(n - 1)^2}{n^3}
	\right]
} =
\lim_{n \rightarrow \infty}{
	\left[
		\dfrac{
			\dfrac{n(n+1)(2n + 1)}{6}
		}
		{
			n^3
		}
	\right]
} = 
\lim_{n \rightarrow \infty}{
	\left[
		\dfrac{2n^3 + 3n^2 + n}{6n^3}
	\right]
} = 
$$
$$
= \lim_{n \rightarrow \infty}{
	\left[
		\dfrac{(2n^3 + 3n^2 + n) : 2n^3}{6n^3 : 2n^3} 
	\right]
} = \dfrac{1}{3}
$$

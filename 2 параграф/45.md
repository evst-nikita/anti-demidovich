Сформулировать с помощью неравенств следующие утверждения:
$$
\mbox{а) } \lim_{n \rightarrow \infty}{x_n} = \infty; \quad \mbox{б) } \lim_{n \rightarrow \infty}{x_n} = -\infty; \quad \mbox{в) } \lim_{n \rightarrow \infty}{x_n} = +\infty.
$$

\medskip

$$
\mbox{а) } \lim_{n \rightarrow \infty}{x_n} = \infty \Leftrightarrow \forall ~E > 0 ~~\exists ~N = N(E) ~~\forall ~n > N: ~|x_n| > E
$$
$$
\mbox{ б) } \lim_{n \rightarrow \infty}{x_n} = -\infty \Leftrightarrow \forall ~E > 0 ~~\exists ~N = N(E) ~~\forall ~n > N: ~x_n < -E
$$
$$
\mbox{в) } \lim_{n \rightarrow \infty}{x_n} = +\infty \Leftrightarrow \forall ~E > 0 ~~\exists ~N = N(E) ~~\forall ~n > N: ~x_n > E
$$

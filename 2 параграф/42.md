№ 42

Доказать, что $x_n$ $(n = 1, 2, \ldots)$ есть бесконечно малая (т.е. имеет предел равный 0), указав для всякого $\varepsilon > 0$ число $N = N(\varepsilon)$ такое, что $|x_n| < \varepsilon$ при $n > N$, если:
$$
\mbox{а) }x_n = \frac{(-1)^{n + 1}}{n}  \quad \mbox{б) } x_n = \frac{2n}{n^3 + 1}
$$
$$
\mbox{в) }x_n = \frac{1}{n!}  \quad \mbox{г) } x_n = (-1)^n \cdot 0,999^n
$$
Для каждого из этих случаев заполнить следующую таблицу:

| $\varepsilon$ | $0,1$ | $0,001$ | $0,0001$ | $\ldots$ |
| ------------- | ----- | ------- | -------- | -------- |
| $N$           |       |         |          |          |
% Не забыть потом переделать её в gpt

\medskip

Для решения подобных задач, будем подставлять $x_n$ в неравенство с модулем, раскрывать, решать неравенство относительно $n$ и подставлять результаты в таблицу

а) $x_n = \frac{(-1)^{n + 1}}{n}$
$$
\left|\frac{(-1)^{n + 1}}{n}\right| < \varepsilon \Leftrightarrow \frac{1}{n} < \varepsilon \Leftrightarrow n > \frac{1}{\varepsilon}
$$

| $\varepsilon$ | $0,1$ | $0,001$ | $0,0001$ | $\ldots$                                         |
| ------------- | ----- | ------- | -------- | ------------------------------------------------ |
| $N$           | $10$  | $1000$  | $10000$  | $\left\lfloor\frac{1}{\varepsilon}\right\rfloor$ |
б) $x_n = \frac{2n}{n^3 + 1}$
$$
\left|\frac{2n}{n^3 + 1}\right| < \varepsilon \Leftrightarrow \frac{2n}{n^3 + 1} < \varepsilon \Leftrightarrow \frac{2n \cdot \frac{1}{n}}{(n^3 + 1) \cdot \frac{1}{n}} < \varepsilon \Leftrightarrow
$$
$$
\Leftrightarrow 0 < \frac{2}{n^2 + \frac{1}{n}} < \frac{2}{n^2} < \varepsilon \Leftrightarrow n^2 > \frac{2}{\varepsilon} \Leftrightarrow n > \sqrt{\frac{2}{\varepsilon}}
$$

| $\varepsilon$ | $0,1$       | $0,001$      | $0,0001$      | $\ldots$                                                |
| ------------- | ----------- | ------------ | ------------- | ------------------------------------------------------- |
| $N$           | $2\sqrt{5}$ | $20\sqrt{5}$ | $100\sqrt{2}$ | $\left\lfloor\sqrt{\frac{2}{\varepsilon}}\right\rfloor$ |
в) $x_n = \frac{2n}{n^3 + 1}$. В решении был выбрал логарифм по основании 10, для удобства работы с таблицой. Почему для сравнения была взята дробь $\frac{1}{2^{n - 1}}$ знали только Бог и сам Демидович. Сейчас же знает только Бог.
$$
\left|\frac{1}{n!}\right| < \varepsilon \Leftrightarrow 0 < \frac{1}{n!} < \frac{1}{2^{n - 1}} < \varepsilon \Leftrightarrow \frac{2}{2^n} < \varepsilon \Leftrightarrow
$$
$$
\Leftrightarrow 2^n > \frac{2}{\varepsilon} \Leftrightarrow \lg{2^n} > lg{\frac{2}{\varepsilon}} \Leftrightarrow n\lg{2} > \lg{2} + \lg{\frac{1}{\varepsilon}} \Leftrightarrow n > 1 + \frac{\lg{\frac{1}{\varepsilon}}}{\lg{2}}
$$

| $\varepsilon$ | $0,1$ | $0,001$ | $0,0001$ | $\ldots$                                                                        |
| ------------- | ----- | ------- | -------- | ------------------------------------------------------------------------------- |
| $N$           | $4$   | $10$    | $14$     | $\left\lfloor\sqrt{1 + \frac{\lg{\frac{1}{\varepsilon}}}{\lg{2}}}\right\rfloor$ |
г) $x_n = (-1)^n \cdot 0,999^n$. В этом доказательстве стоит учесть, что $\lg{0.999} < 0$.
$$
|(-1)^n \cdot 0,999^n| < \varepsilon \Leftrightarrow 0,999^n < \varepsilon \Leftrightarrow n\lg{0.999} < \lg{\varepsilon} \Leftrightarrow
$$
$$
\Leftrightarrow n > \frac{\lg{\varepsilon}}{\lg{0.999}} \approx2330 \lg{\frac{1}{\varepsilon}}
$$

| $\varepsilon$ | $0,1$  | $0,001$ | $0,0001$ | $\ldots$                                                   |
| ------------- | ------ | ------- | -------- | ---------------------------------------------------------- |
| $N$           | $2330$ | $6990$  | $9320$   | $\left\lfloor2330 \lg{\frac{1}{\varepsilon}}\right\rfloor$ |

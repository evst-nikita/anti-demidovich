№ 26

$$
|x + 2| + |x - 2| \leq 12
$$

\medskip

Первым делом рассчитаем точки, где выражения в модулях меняют знак:
$$
x + 2 = 0 \Leftrightarrow x = -2 \quad x - 2 = 0 \Leftrightarrow x = 2
$$
Теперь рассмотрим неравенство в виде системы:
$$
|x + 2| + |x - 2| \leq 12 = \begin{cases}
-x - 2 + (-x + 2) \leq 12, & x \leq -2\\
x + 2 + (-x + 2) \leq 12, & -2 < x \leq 2\\
x + 2 + x - 2 \leq 12, & x > 2
\end{cases} =
$$
$$
= \begin{cases}
x \geq -6, & x \leq -2\\
4 \leq 12, & -2 < x \leq 2\\
x \leq 6, & x > 2
\end{cases}
$$
Рассмотрим полученные неравенства:
\begin{itemize}
\item В первом у нас получается пересечение двух неравенств: полученное неравенство $x \geq -6$ и неравенство из области определения $x \leq -2$. Итого получаем:  $-6 \leq x \leq -2$.
\item Во втором у нас получилось, что неравенство верно на всей области определения: $2 < x \leq 2$.
\item В третьем случае у нас пересечение неравенств дает нам: $2 < x \leq 6$.
\end{itemize}
Отсюда делаем вывод: неравенство верно при: 

$x \in [-6, ~-2] ~ \bigcup ~ (-2, ~2] ~ \bigcup ~ (2, ~6] = -6 \leq x \leq 6 = |x| \leq 6$.

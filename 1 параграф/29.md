$$
|x(1 - x)| < 0.05
$$

\medskip

По свойству модулей:
$$
|x(1 - x)| < 0.05 \Leftrightarrow |x||1 - x| < 0.05
$$
Первым делом рассчитаем точки, где выражения в модулях меняют знак:
$$
1 - x = 0 \Leftrightarrow x = 1 \quad x = 0
$$
Теперь рассмотрим неравенство в виде системы:
$$
|x||1 - x| < 0.05 = \begin{cases}
-x(1 - x) < 0.05, & x \leq 0\\
x(1 - x) < 0.05, & 0 < x \leq 1\\
x(x - 1) < 0.05, & x > 1
\end{cases} =
$$
$$
= \begin{cases}
x^2 - x - 0.05 < 0, & x \leq 0\\
x^2 - x + 0.05 > 0, & 0 < x \leq 1\\
x^2 - x - 0.05 < 0, & x > 1
\end{cases} = \begin{cases}
\left(x - \frac{1 + \sqrt{1.2}}{2} \right)\left(x - \frac{1 - \sqrt{1.2}}{2} \right) < 0, & x \leq 0\\
\left(x - \frac{1 + \sqrt{0.8}}{2} \right)\left(x - \frac{1 - \sqrt{0.8}}{2} \right) > 0, & 0 < x \leq 1\\
\left(x - \frac{1 + \sqrt{1.2}}{2} \right)\left(x - \frac{1 - \sqrt{1.2}}{2} \right) < 0, & x > 1
\end{cases}
$$
Рассмотрим полученные неравенства:
\begin{enumerate}
\item Парабола меньше нуля на промежутке $x \in \left(\frac{1 - \sqrt{1.2}}{2}, ~\frac{1 + \sqrt{1.2}}{2}\right)$.  Пересечение неравенств даёт нам область: $\frac{1 - \sqrt{1.2}}{2} < x \leq 0$.
\item Парабола больше нуля на промежутке: $x \in \left(-\infty, ~\frac{1 - \sqrt{0.8}}{2}\right) ~\bigcup~ \left( \frac{1 + \sqrt{0.8}}{2}, ~+\infty\right)$. Пересечение неравенств даёт нам область: $x \in \left(0, ~\frac{1 - \sqrt{0.8}}{2}\right) ~\bigcup~ \left( \frac{1 + \sqrt{0.8}}{2}, ~1\right]$.
\item В третьем случае парабола меньше нуля на промежутке $x \in \left(\frac{1 - \sqrt{1.2}}{2}, ~\frac{1 + \sqrt{1.2}}{2}\right)$.  Пересечение неравенств даёт нам область: $1 < x < \frac{1 + \sqrt{1.2}}{2}$.
\end{enumerate}
Отсюда делаем вывод: неравенство верно при 

$$
x \in \left(\frac{1 - \sqrt{1.2}}{2}, ~0\right] ~ \bigcup ~ \left(0, ~\frac{1 - \sqrt{0.8}}{2}\right) ~\bigcup~ \left( \frac{1 + \sqrt{0.8}}{2}, ~1\right] ~\bigcup~ \left(1, ~\frac{1 + \sqrt{1.2}}{2}\right)
$$
Сократим запись:
$$
x \in \left(\frac{1 - \sqrt{1.2}}{2}, ~\frac{1 - \sqrt{0.8}}{2}\right) ~\bigcup~ \left( \frac{1 + \sqrt{0.8}}{2}, ~\frac{1 + \sqrt{1.2}}{2}\right)
$$
Осталось подогнать это под ответ в учебнике: Умножим числитель и знаменатель каждой дроби на 5 и запишем в форме неравенства
$$
\frac{5 - \sqrt{30}}{2} < x < \frac{5 - \sqrt{20}}{2}; \quad \frac{5 + \sqrt{20}}{2} < x < \frac{5 + \sqrt{30}}{2}
$$

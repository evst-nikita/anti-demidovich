№ 24
$$
|x| > |x + 1|
$$

\medskip

Тут надо взять точки, при которых выражения внутри модулей меняют знак и раскрыть их для каждой части прямой. В данном уравнении это точки $x = 0$ и $x = -1$:
$$
|x| > |x + 1| = \begin{cases}
	-x > -x - 1, & x \leq -1\\
	-x > x + 1, & -1 < x \leq 0\\
	x > x + 1, & x > 0
\end{cases} =
$$
$$
= \begin{cases}
	0 > -1, & x \leq -1\\
	-2x > 1, & -1 < x \leq 0\\
	0 > 1, & x > 0
\end{cases} = 
\begin{cases}
	0 > -1, & x \leq -1\\
	x < -\frac{1}{2}, & -1 < x \leq 0\\
	0 > 1, & x > 0
\end{cases}
$$
Рассмотрим полученные неравенства:
\begin{itemize}
\item В первом у нас получилось верное неравенство, а значит оно работает для всех $x \leq -1$.
\item Во втором у нас получилось, что неравенство верно только когда $x < -\frac{1}{2}$.
\item В третьем случае у нас получилось неверное неравенство, а значит при $x > 0$ оно всегда будет неверно. 
\end{itemize}
Отсюда делаем вывод: неравенство верно при: $x \in (-\infty, ~-1] ~ \bigcup ~ (-1, ~-\frac{1}{2}) = x < -\frac{1}{2}$.

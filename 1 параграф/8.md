Доказать неравенство:
$$
n! < \left(\frac{n + 1}{2}\right)^n \mbox{    (n > 1)}
$$
Указание: использовать неравенство
$$
\left(\frac{n + 2}{n + 1}\right)^{n + 1} = \left(1 + \frac{1}{n + 1}\right)^{n + 1} > 2 \quad (n = 1, 2, \ldots)
$$
---
$$
\mbox{База индукции: } 2! = 2 < \left(\frac{3}{2}\right)^2 = \frac{9}{4} = 2.25
$$
Докажем, что: $(n + 1)! < \left(\frac{n + 2}{2}\right)^{n + 1}$

$$

$$
\left(\frac{n + 2}{n + 1}\right)^{n + 1} > 2 \Leftrightarrow \left(\frac{n + 1}{n + 2}\right)^{n + 1} < \frac{1}{2}
$$
$$
(=) \frac{(n + 1)^{n + 1} (n + 2)^{n + 1}}{2^n (n + 2)^{n + 1}} = \frac{(n + 2)^{n + 1}}{2^n} \cdot \left(\frac{n + 1}{n + 2}\right)^{n + 1} <

$$

Доказать, что если x > -1, то справедливо неравенство:
$$
(1 + x)^n \geq 1 + nx \mbox{    (n > 1)}
$$
причем знак равенства имеет место быть лишь при x = 0

Докажем, что: $(1 + x)^{n + 1} \geq 1 + (n + 1)x = 1 + nx + x$

$$
Так как $nx^2$ положительное число, то справедливо неравенство
$$
\geq 1 + nx + x = 1 + (n + 1)x
$$
ЧТД. Мат. индукция доказана. Теперь докажем, что знак равенства будет только когда x = 0 с помощью бинома Ньютона
$$
(1 + x)^n = 1 + nx \Leftrightarrow \sum_{m = 0}^{n}C^m_n x^m = 1 + nx \Leftrightarrow 1 + \sum_{m = 1}^{n}C^m_n x^m = 1 + nx
$$
$$
\sum_{m = 1}^{n}C^m_n x^m = nx \Leftrightarrow \sum_{m = 1}^{n}C^m_n x^{m - 1} = n \Leftrightarrow \sum_{m = 1}^{n}\frac{n!}{m!(n - m)!} x^{m - 1} = n
$$
$$
\sum_{m = 1}^{n}\frac{(n-1)!}{m!(n - m)!} x^{m - 1} = 1 \Leftrightarrow \frac{(n - 1)!}{1!(n - 1)!} + \sum_{m = 2}^{n}\frac{(n - 1)!}{m!(n - m)!} x^{m - 1} = 1
$$
$$
1 + \sum_{m = 2}^{n}\frac{(n-1)!}{m!(n - m)!} x^{m - 1} = 1 \Leftrightarrow \sum_{m = 2}^{n}\frac{(n - 1)!}{m!(n - m)!} x^{m - 1} = 0
$$
$C^m_n$ всегда дает натуральное число, так что единственным вариантом, при котором вся сумма будет равна нулю при любом n > 1 остается случай, когда x = 0

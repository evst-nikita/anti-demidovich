№ 28

$$
||x + 1| - |x - 1|| < 1
$$

\medskip

Дабы упростить решение неравенства, возведем его обе части в квадрат, чтобы избавиться от модуля:
$$
(|x + 1| - |x -1|)^2 < 1 \Rightarrow (x + 1)^2 - 2|x+1||x-1| + (x - 1)^2 < 1 \Leftrightarrow
$$
$$
\Leftrightarrow 2x^2 + 1 - 2|x + 1||x - 1| < 0
$$
Рассчитаем точки, где выражения в модулях меняют знак:
$$
x = 1 \quad x = -1
$$
Теперь рассмотрим неравенство в виде системы:
$$
2x^2 + 1 - 2|x + 1||x - 1| < 0 = \begin{cases}
2x^2 + 1 - 2(x^2 - 1) < 0, & x \leq -1\\
2x^2 + 1 + 2(x^2 - 1) < 0, & -1 < x \leq 1\\
2x^2 + 1 - 2(x^2 - 1) < 0, & x > 1
\end{cases} =
$$
$$
= \begin{cases}
3 < 0, & x \leq -1\\
x^2 < \frac{1}{4}, & -1 < x \leq 1\\
3 < 0, & x > 1
\end{cases}
$$
Рассмотрим полученные неравенства:
\begin{enumerate}
\item Неравенство неверно. В данной области определения решений нет.
\item $x^2 < \frac{1}{4}$ раскрывается, как $-\frac{1}{2} < x < \frac{1}{2}$. Пересечение неравенств дает нам этот же самый результат.
\item Неравенство неверно. В данной области определения решений нет..
\end{enumerate}
Отсюда делаем вывод: неравенство верно при: 

$x \in (-\frac{1}{2}, ~\frac{1}{2})= -\frac{1}{2} < x < \frac{1}{2}$.

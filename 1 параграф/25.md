№ 25

$$
|2x - 1| < |x - 1|
$$

\medskip

Первым делом рассчитаем точки, где выражения в модулях меняют знак:
$$
2x - 1 = 0 \Leftrightarrow x = \frac{1}{2} \quad x - 1 = 0 \Leftrightarrow x = 1
$$
Теперь рассмотрим неравенство в виде системы:
$$
|2x - 1| < |x - 1| = \begin{cases}
-2x + 1 < -x + 1, & x \leq \frac{1}{2}\\
2x - 1 < -x + 1, & \frac{1}{2} < x \leq 1\\
2x - 1 < x - 1, & 1 < x
\end{cases} =
$$
$$
= \begin{cases}
x > 0, & x \leq \frac{1}{2}\\
x < \frac{2}{3}, & \frac{1}{2} < x \leq 1\\
x < 0, & 1 < x
\end{cases}
$$
Рассмотрим полученные неравенства:
\begin{itemize}
\item В первом у нас получается пересечение двух неравенств: полученное неравенство $x > 0$ и неравенство по условию $x \leq \frac{1}{2}$. Итого получаем:  $0 < x < \frac{1}{2}$.
\item Во втором у нас получилось, что неравенство верно, когда $x < \frac{2}{3}$. Делаем пересечение с неравенством по области определения и получаем: $\frac{1}{2} < x < \frac{2}{3}$.
\item В третьем случае у нас происходит несовпадение неравенств и их пересечение дает пустое множество.
\end{itemize}
Отсюда делаем вывод: неравенство верно при: $x \in (0, ~\frac{1}{2}] ~ \bigcup ~ (\frac{1}{2}, ~\frac{2}{3}) = 0 < x < \frac{2}{3}$.

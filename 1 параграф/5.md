$$
\mbox{Пусть } a^{[n]} = a(a - h) \ldots [a - (n - 1)h] \mbox{ и } a^{[0]} = 1
$$
$$
\mbox{Доказать, что } (a + b)^{[n]} = \sum_{m = 0}^{n} C^m_n a^{[n - m]} b^{[m]}
$$ 
Где $C^m_n$ - число сочетаний из n элементов по m. Вывести отсюда формулу Бинома Ньютона.

---
$$
\mbox{База индукции: } (a + b)^{[1]} = (a + b) = a + b
$$
$$
\sum_{m = 0}^{1}C^m_n a^{[n - m]} b^{[m]} = C^0_1 a^{[1]} b^{[0]} + C^1_1 a^{[0]} b^{[1]} = 1 \cdot a \cdot 1 + 1 \cdot 1 \cdot b = a + b
$$
Докажем, что: $(a + b)^{[n + 1]} = \sum_{m = 0}^{n + 1}C^m_{n + 1} a^{[n + 1 - m]} b^{[m]}$
По предположению математической индукции: (Удачи)
$$
(a + b)^{[n + 1]} = (a + b)^{[n]} (a + b - nh) = \sum_{m = 0}^{n}C^m_n a^{[n - m]} b^{[m]} \cdot (a + b - nh) =
$$
$$
= (C^0_n a^{[n]} b^{[0]} + C^1_n a^{[n - 1]} b^{[1]} + \ldots + C^{n - 1}_n a^{[1]} b^{[n - 1]} + C^n_n a^{[0]} b^{[n]}) \cdot (a + b - nh) = 
$$
$$
= C^0_n a^{[n]} b^{[0]} (a + b - nh) + C^1_n a^{[n - 1]} b^{[1]} (a + b - nh) + \ldots + C^{n - 1}_n a^{[1]} b^{[n - 1]} (a + b - nh) + 
$$
$$
+ C^n_n a^{[0]} b^{[n]} (a + b - nh) = 
$$
$$

$$
$$

$$
$$
= C^0_n a^{[n]} (a - nh) b^{[0]} + C^0_n a^{[n]} b^{[0]} b + C^1_n a^{[n - 1]} (a - (n - 1)h) b^{[1]} + C^1_n a^{[n - 1]} b^{[1]} (b - h) + \ldots +

Рассмотрим пару вспомогательных моментов:
$$
\mbox{Формула $C^m_n$: } C^m_n = \frac{n!}{m!(n - m)!}
$$
Части вида $C^0_n + C^1_n$ или $C^{n - 1}_n + C^n_n$ можно обобщить в виде: $C^{m - 1}_n + C^m_n$
Вычислим сумму сочетаний:
$$
C^{m - 1}_n + C^m_n = \frac{n!}{(m - 1)!(n - m + 1)!} + \frac{n!}{m!(n - m)!} = \frac{n!m + n!(n + 1 - m)}{m!(n + 1 - m)!} = 
$$
$$
= \frac{n!(m + n + 1 - m)}{m!(n + 1 - m)!} = \frac{n!(n + 1)}{m!(n + 1 - m)!} = \frac{(n + 1)!}{m!(n + 1 - m)!} = C^m_{n + 1}
$$
Также важно заметить (для замены последнего члена), что сочетание имеет интересное свойство: $C^a_a = C^b_b = 1$
То есть $C^n_n = C^{n + 1}_{n + 1}$
Аналогично с $C^0_a = C^0_b = 1$, то есть $C^0_n = C^0_{n + 1}$. Это нам пригодится для первого члена.
Закончим решение:
$$
(=) C^0_{n + 1}a^{[n + 1]} b^{[0]} + C^1_{n + 1} a^{[1]} b^{[1]} + \ldots + C^n_{n + 1} a^{[1]} b^{[n]} +
$$
$$
C^{n + 1}_{n + 1}a^{[0]} b^{[n + 1]} = \sum_{m = 0}^{n + 1} C^m_{n + 1} a^{[n + 1]} b^{[m]}
$$
ЧТД. Формула Бинома Ньютона является частным случаем этой формулы, когда $h = 0$
$$
\mbox{Бином Ньютона: } (a + b)^n = \sum_{k = 0}^{n} C^k_n a^{n - m} b^m
$$
